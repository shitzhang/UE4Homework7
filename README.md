# Tencent UE4 Class Homework 7  

打开Server文件夹里的 prototypeServer.exe - P_TestMap 开启专属服务器  

客户端进入游戏点击Play进入房间  

游戏时长设置为2分钟  

玩家死后有布娃娃效果，进入观战模式，5秒后重生  

结算界面包括分数，击杀数，死亡数  


## 基本操作  
移动：WASD 跳：Space 蹲：左Ctrl 冲刺：左Shift 开火：鼠标左键 瞄准：鼠标右键  
切换武器：鼠标滚轮 键盘1，2  
Drop：G  
Pick：E  
切换近战：X  
自由视角：左Alt
